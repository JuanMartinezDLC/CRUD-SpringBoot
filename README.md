# CRUD-SpringBoot
Este proyecto es un CRUD (Create, Read, Update, Delete) hecho con Spring Boot como parte de mi aprendizaje en desarrollo web. Es un ejemplo funcional para entender cómo construir servicios web con Java y Spring Boot.

No olvides cambiar en la ruta src/main/resources en application-example.properties tus datos de tu zona horaria, tu usuario y tu contraseña
